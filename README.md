# Clean VPN

Android client for [opencfg.xyz](https://opencfg.xyz). Supports SSH, SSL, Direct, Proxy, and V2Ray configs.

## Features

- SSH tunneling
- SSL/TLS tunneling
- Direct proxy connection
- HTTP/SOCKS proxy
- V2Ray (VMess / VLESS)
- Import config via file or QR code
- Multiple server profiles

## Installation

Download the APK from [Releases](https://github.com/YOUR-USERNAME/YOUR-REPO/releases), then on your phone enable **Install from unknown sources** (Settings → Apps → Special access → Install unknown apps) and open the downloaded file.

Only download from this repo's Releases page, not from third-party mirrors.

## Usage

Get your config from your [opencfg.xyz](https://opencfg.xyz) account, then in the app tap **Add Profile** and import it (file or QR code), then **Connect**.

## Requirements

Android 7.0 or higher, plus an active config from opencfg.xyz.
